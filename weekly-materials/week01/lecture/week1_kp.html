Data Science#

What is it?#

Learning from Data

  • Large collection of (electronic) data + programming + statistics

  • A lot of data science tools and resources are free to use! (open-source)

What is code?#

  • A highly-structured set of instructions (program, or code)

  • in a language that a computer can understand (programming/coding language)

What do I mean by ‚Äúunderstand‚Äù?

  • The computer can convert the code into a sequential set of actions that it can run (execute).

  • For example: setting an alarm.

  • For example: performing calculations

These instructions need to be written in a special programming language (not natural languages, like English or French)

  • There are several programming languages (C, Java, Python)

  • Python has a very friendly syntax (the grammar of the language), and many many inbuilt tools that are useful for data science

What you‚Äôll learn in this course#

How to write code (instructions) that can read data and compute various statistical metrics of interest from it.

Data#

The data that we work with is usually information collected by other scientists and researchers over years and decades, and is stored electronically in a variety of formats: text files, excel files, images, videos.

For example, I can obtain a large CSV file from https://weather.gc.ca/ (Environment and Weather Canada) that contains the daily max temperature recorded in Toronto from when they first started recording it ‚Äì I think 1840 ‚Äì in a spreadsheet.

Science#

Say I‚Äôm interested in a bunch of questions around global warming. We can write code that will

  • read this data file

  • tell us how much the average temperature each month increased from 1840 to 2023

  • find which year had the hottest March 15, or the coldest January 10th

  • plot all of this in beautiful graphs.

Where do we write and run code?#

  • Typically, one writes code in a text file that is saved as <name>.py (similar to .pdf, or .docx), and the file is then run (or executed) in a terminal (phew!).

  • Here, we use Jupyter Notebooks, a software that runs in your browser and allows you write, run, and view the outputs of your code ‚Äì along with normal text, like a word document.

  • You can access Jupyter by going to https://jupyter.utoronto.ca (see the JupyterHub guide on the course website).

Jupyter Notebooks#

Allow you write (a)code in Python, as well as (b)notes (explanations, figures) in Markdown.

  • Run the code in the browser

  • Display the outputs in a nice format in the browser
